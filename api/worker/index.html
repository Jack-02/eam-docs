    员工¶

    注意

    不加特殊说明，所有请求都需要携带包含 sessionId 字段的 cookie。

    资产查看¶

    群体资产查看¶

    • [ POST ]    /api/asset/view

    员工分页查询自己名下挂账了哪些资产以及部门内未被领用的资产，按创建时间倒序排序。

    示例 
    {
    "method":1,
    "name":"",
}

    {
    "code": 0,
    "info": "Succeed searching the asset",
    "data": [
        {
            id: number;
            name: string;
            price: number;
            number: number;
            life: number;
        }
    ]
}

    请求参数¶

    参数 类型 说明
    method const number = ½ 分类参数，区别查询的资产，1为员工治下资产，2为部门内未领用资产
    name str 查询的资产名称（模糊查找）
    page int 查询页数
    pagesize int 页面大小

    响应状态¶

    状态码 说明
    0 查询成功
    2 分页错误

    响应数据¶

    字段 类型 说明
    data array of [objects] 查询得到的资产
    totalsize int 全部资产的个数
    objects数据 类型 说明
    id int 资产id
    name str 资产名称
    price number 资产价格
    life number 折旧年限
    number number 资产数目

    单个资产查看¶

    • [ POST ]    /api/asset/view

    员工可以查看资产的详情信息，对于资产的描述图片启用晚加载机制。

    示例 
    {
    "method":3,
    "asser_id":1,
}

    {
    "code": 0, 
    "info": "Succeed geting asset info", 
    "data": {
        "id": 417027, "parent": null, "parent_id": null, "name": "asset1", "department": "\u8ba1\u7b97\u673a\u7cfb", "department_id": 41, "classname": "\u7535\u5668", "classcategory": 1, "property": "", "user": "gll", "price": "1000.00", "life": null, "value": "2000.00", "number": 2, "description": null, "img": null, "position": "", "condition": 1, "borrower": "wyk", "purchase_time": 1684605622.006234, "depreciation_model": null, "alarm_time": null, "scrap_time": null, "endvalue": null
    }
}

    请求参数¶

    参数 类型 说明
    method const number = 3 分类参数，区别查询的资产，3为单个资产查询
    asset_id number 查询的资产id

    响应状态¶

    状态码 说明
    0 查询成功
    1 查询失败，id无效

    响应数据¶

    字段 类型 说明
    id int 资产id
    name str 资产名称
    price number 资产价格
    life number 折旧年限
    number number 资产数目
    alarm_time number 告警时间
    borrower str 资产借用者
    classcategory number=½ 资产类别，1为数量型2为条目型
    classname str 资产类型
    condition number 该资产为借出借入维保退库等状态参数
    department str 资产所属部门
    department_id number 资产所属部门id
    endvalue number 折旧后终值
    img picture 资产图片
    life number 资产寿命
    value number 当前净值
    position str 当前位置
    purchase_time number 购买时间
    user str 资产管理员名称

    资产领用，退库，维保¶

    • [ POST ]    /api/asset/apply

    员工向资产管理员领用资产。一次可以领用多个资产。员工先提交领用申请，经过资产管理员审批后，资产会挂账到当前员工名下。

    示例 
    {
    "method": 1/2/3, 
    "asset_ids":[1,2,3,...]
}

    {
    "code": 0,
    "info": "Succeed apply for/return/maintain asset(s)",
}

    请求参数¶

    参数 类型 说明
    method const number = ½/3 //½/3分别代表领用、退库、维保
    asset_ids array of number [1,2,3,...]操作资产的id

    响应状态¶

    状态码 说明
    0 提交成功
    2 提交失败，无操作权限
    3 提交失败，资产不存在
    4 提交失败，资产不可用
    5 提交失败，该资产不可重复申请

    响应数据¶

    字段 类型 说明
    info str 成功领用/退库/维保资产

    资产转移¶

    查询被转移的员工¶

    • [ POST ]    api/worker/filter

    查询同一实体内不同部门下的员工，支持id、部门名称（模糊）、用户名称（模糊）的搜索，用于资产转移目标

    示例 
    {
    "id": 1,
    "name": "xxx", 
    "nickname": "xxx", 
    "department": "xxx",
}

    {
    "code": 0,
    "info": "Get related all users",
    "data": [
        {
            "user_id": 1,
            "name": "xxx",
            "nickname": "xxx",
            "department": "xxx",
        }
    ]
}

    请求参数¶

    参数 类型 说明
    id number 查询用户id
    name str 查询用户名称（模糊）
    nickname str 查询用户昵称（模糊）
    department str 查询用户部门（模糊）

    响应状态¶

    状态码 说明
    0 查询成功
    1 查询失败

    响应数据¶

    字段 类型 说明
    data array of objects 数据数组
    objects 类型 说明
    user_id int 用户id
    name str 用户名称
    nickname str 用户昵称
    department str 部门名称

    进行资产转移¶

    • [ POST ]    /api/asset/transfer

    员工将某些资产转移给另一个员工。资产转移提交申请后，经过主管审批通过后，资产的挂账人和部门切换为新的员工和部门。资产的位置也跟着迁移到新位置。

    示例 
    {
    "target_worker_id": 222,
    "asset_ids":[123,344]
}

    {
    "code": 0,
    "info": "Succeed transfer asset(s)",
}

    请求参数¶

    参数 类型 说明
    target_worker_id number 转至员工的id
    asset_ids array of number [1,2,3,...]转移资产的id

    响应状态¶

    状态码 说明
    0 转移成功
    -1 转移失败，目标用户不属于该企业实体或身份不为员工
    1 转移失败，目标用户不存在
    2 转移失败，没有权限
    3 转移失败，资产不存在
    4 转移失败，资产不可执行转移操作
    5 提交失败，该资产不可重复申请

    响应数据¶

    字段 类型 说明
    info str 成功转移资产
